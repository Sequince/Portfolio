Time Watch:
   -   Technologies Used:   AutoHotKey


Description:

This project assists the user in getting away from the computer throughout the day using popup reminders.

Once the daily setting is selected, the 'game' pops up reminders as set intervals using a battle screen.

The user can close the popup(snooze) or click battle. If enough walking around breaks(battle screens) were completed in a day the user will level up come the next day.
